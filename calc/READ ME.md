# CLI Calculator
## Features
- Basic math operations
- Last result variable (`r`)
- Command history
- Clear screen support
- No unsafe `eval()` usage

## Usage
```
> 2 + 2
= 4
> r * 3
= 12
> history
1. 2 + 2 = 4
2. 4 * 3 = 12
```

## Why?
- Safer than calculators using `eval()`
- Demonstrates parsing/lexing algorithms
- Useful CLI tool
- For fun!
